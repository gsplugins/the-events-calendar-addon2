Display events from The Events Calendar in grid, list, table, slider, masonry, and calendar layouts.

== Description ==

The Events Calendar Addon helps you showcase events from The Events Calendar plugin using beautiful layouts, shortcode builder, and flexible display settings.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/the-events-calendar-addon2/` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin.
3. Create or select a shortcode from the shortcode builder.
4. Display it on any page using the generated shortcode.

== Frequently Asked Questions ==

= Does this plugin require The Events Calendar? =

Yes. This plugin is designed to display events from The Events Calendar.
